# Productivity OS 🚀

A gamified productivity and habit tracking platform built with React + Vite.

## Features

✅ Habit tracking  
⭐ XP and level system  
🔥 Daily streak tracking  
🏆 Achievement badges  
😊 Mood tracker  
📊 AI-powered weekly insights  
🟩 Yearly habit heatmap  
📈 Progress analytics  
🎉 XP burst animations  
🚀 Level-up system with colored badges  
💾 Local storage support  
🌙 Modern glassmorphism UI

## Built With

- React
- Vite
- Tailwind CSS
- Recharts
- LocalStorage API

Designed to make self-improvement feel engaging and fun.
